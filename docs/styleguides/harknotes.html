Foundation

Elevation

Depth comes from background lightness and border strength — not shadows. Shadows are reserved for floating elements. Each level sits on the level below it.

Level 0 Background Main app window canvas --bg
Level 1 Elevated Sidebar, toolbar panels --bg-elevated
Level 2 Surface Cards, list items --surface-1
Level 3 Floating Dropdowns, popovers --surface-2 + shadow

Guidelines

Motion

Motion communicates state. It should never draw attention to itself — if you notice the animation, it's too much.

Easing & Duration

--ease-out cubic-bezier(0.16,1,0.3,1) Enter animations — fast start, eases to rest
--ease-in cubic-bezier(0.4,0,1,1) Exit animations — slow start, fast exit
--ease-inout cubic-bezier(0.65,0,0.35,1) Transforms, position changes
120ms Hover, colour transitions, micro-feedback
200ms Toggles, expanding elements

Guidelines

Voice & Tone

Harknotes is a professional tool. The copy should be invisible — direct, precise, honest. It never cheers or apologises.

✓ Write like this
"Notes saved to Meetings/2026-03-02.md" "18h of 20h used this month" "Transcribing with whisper.cpp" "Stop recording" "Transcription failed — audio device not found"
✗ Not like this
"Your notes are ready! 🎉" "Uh oh! You're almost out of time" "Our magical AI is working hard…" "Click here to finish" "Oops! Something went wrong"
Direct.

Say what happened. Skip emotional framing.

Precise.

Use numbers, file paths, durations. Never "some" or "a few".

Quiet.

Good copy disappears. You only notice bad copy.

Honest.

When something fails, say what failed and why.
