How to export
Google / AppleUse variant A or C. Right-click the 220px square → "Open in new tab" → save → upload. Platform circles it.
GitHub orgUse variant B. Same method. GitHub rounds with its own radius.
High-res PNGView source → copy the outer <svg> → paste into Figma or Inkscape → export at 512×512.
FaviconExport dark variant at 32×32, convert to .ico. At 16px the opacity layering simplifies but still reads.
